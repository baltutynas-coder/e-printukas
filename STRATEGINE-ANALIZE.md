# e.printukas.lt — Strateginė analizė ir redesign planas

**Data:** 2026-04-23  
**Tikslas:** Pozicionuoti e.printukas.lt kaip drąsiausią, moderniausią ir lengviausią naudoti B2B reklaminio tekstilės parduotuvę Lietuvoje — pasaulinio lygio UX/dizainu.

---

## TL;DR — vienos minutės santrauka

**Rinkoje yra "tuščia erdvė"**: Lietuvos B2B reklaminės tekstilės konkurentai (drabuziaireklamai.lt, grazugrazu.lt, ago.lt, adversta.lt) — stiprūs produktais, bet **silpni dizainu ir UX**. Jų svetainės atrodo 2015 metų. Tavo strateginis pranašumas — **pasaulinio lygio dizainas kaip rinkos pareiškimas**.

**Tris strateginiai sprendimai:**

1. **Min. užsakymas — 10 vnt (NE 50 vnt / 200 €)**. Rinkos standartas. Tavo anksčiau sugalvotas 200 € / 50 vnt slenkstis atstumtų 70% B2B klientų (maži biurai, renginiai, pradedantys verslai).

2. **B2B modelis = „glide pricing"** (pakopinė kainodara). Kiekvienas papildomas vnt — mažesnė kaina. Tai 4imprint pagrindinis USP ir tarptautinis standartas.

3. **Kategorijų struktūra = Lytis × Tipas × Proga** (grazugrazu.lt principas), + atskira **"Dovanos ir aksesuarai"** šaka (Stamina produktams).

## 2. Tarptautinis etalonas — 4imprint

4imprint (FTSE 250 listed, Š. Amerikos rinkos lyderis, 30 000+ produktų) naudoja **10 B2B funkcijų**, kurių LT konkurentai beveik nėra. Tai mūsų „copy-from-the-best" sąrašas:

1. **Glide pricing™** — kaina mažėja automatiškai pagal užsakomą kiekį. Vaizduojama kaip lentelė prie produkto:
   ```
   10–24 vnt    6,50 €/vnt
   25–49 vnt    5,90 €/vnt
   50–99 vnt    5,40 €/vnt
   100+ vnt     4,90 €/vnt
   ```
2. **Min. order — 6 vnt** (mažas barjeras).
3. **Pristatymo laiko filtras**: „24 h / 2 d / 3 d / 5 d" — klientas gali filtruoti produktus pagal skubą.
4. **Spaudos spalvų skaičius kaip filtras** (1 spalva / 2+ / 3+…).
5. **Spaudos vietų skaičius kaip filtras** (1 / 2+ / 3+).
6. **Nemokami pavyzdžiai** — klientas gauna fizinį produktą prieš užsakymą.
7. **Nemokama dizaino pagalba** — art team konsultuoja nemokamai.
8. **Produktų atsiliepimai** — social proof.
9. **„No surprise pricing"** — jokio papildomo mokesčio už dydžius XL+, siuvinėjimo stygų skaičių, etc.
10. **Blog / educational content** — thought leadership + SEO.

### Apdovanojimus laimėjusios e-commerce svetainės (Awwwards 2025)

- **E-commerce Site of the Year 2025: Scout Motors** — "born from an American legend, retooled for a new era" — **storytelling + scroll animations**. Kiekvienam produktui yra istorija.
- **TRUEWERK** (Honorable Mention Mar 2026) — workwear brand, **techniška pateiktis** (medžiagos, veikimas, spec sheet).
- **Chrome Industries**, **DROPDEAD** — drąsios tipografijos, bold brand, judanti hero sekcija.

**Ką pasiimti:** hero su video/scroll animacija, produktų pateikimas su „istorija" (ne tik SKU), mikrointerakcijos (hover, scroll).

## 6. B2B kainodaros modelis — „glide pricing"

Kiekvienas produktas turi **5 kainų lygius**, saugomus DB. Sistema automatiškai parenka kainą pagal krepšelio kiekį.

### Pavyzdys — klasikinis marškinėlis

| Kiekis | Kaina už vnt | Nuolaida | Sutaupysite |
|---|---|---|---|
| **1–9 vnt** | 9,50 € | – | – |
| **10–24 vnt** | 6,90 € | −27% | 52 € (už 20 vnt) |
| **25–49 vnt** | 5,90 € | −38% | 180 € (už 50 vnt) |
| **50–99 vnt** | 5,40 € | −43% | 410 € (už 100 vnt) |
| **100+ vnt** | 4,90 € | −48% | ... |

### Kaip tai rodoma produkto puslapyje

Prie kainos — interaktyvi lentelė su slankikliu:
```
[Slankiklis: 10 → 100 vnt]
Dabartinė kaina: 6,90 €/vnt × 20 = 138,00 €
Sutaupėte: 52,00 € (vs. 1 vnt kaina)
```

### DB schema (papildymas esamai)

```prisma
model ProductPriceTier {
  id         String  @id @default(cuid())
  productId  String
  product    Product @relation(fields: [productId], references: [id])
  minQty     Int     // nuo kiek vnt galioja
  maxQty     Int?    // iki kiek (null = „ir daugiau")
  pricePerUnit Decimal @db.Decimal(10, 2)
  
  @@index([productId, minQty])
}
```

Migracija: `npx prisma migrate dev --name add_price_tiers`.

### Spaudos ir siuvinėjimo kaina

Atskira papildomų paslaugų lentelė:
```
Spauda (viena vieta, 1 spalva):   +1,50 €/vnt
Spauda (viena vieta, 2–3 spalvos): +2,50 €/vnt
Spauda (DTF, spalvota):           +3,00 €/vnt
Siuvinėjimas (iki 5000 stygų):    +3,50 €/vnt
Siuvinėjimas (5000+ stygų):       +4,50 €/vnt
```

Check-out metu — automatinis skaičiavimas server-side.

2. **Didelė drąsi tipografija** (Space Grotesk, Archivo, Inter Display)

## 10. Accessibility (WCAG 2.1 AA) — konkurentų pranokimas

LT konkurentai WCAG **NEATITINKA**. Tu gali būti pirmas. Tai ne tik etika — tai **reikalavimas** dirbant su įmonėmis ir valstybės sektoriumi (pvz., savivaldybės, universitetai).

### Minimalūs reikalavimai

- [ ] Spalvų kontrastas **4,5:1** tekstui, **3:1** UI elementams
- [ ] Visos interakcijos galimos klaviatūra (Tab, Enter, Space, Esc)
- [ ] `aria-label`, `aria-labelledby`, `aria-describedby` kur reikia
- [ ] `<button>` vs `<a>` — semantiškai tvarkinga
- [ ] Focus ring visur matomas (ne `outline: none` be alternatyvos!)
- [ ] Paveikslėliai — `alt` atributas (nebūtinas dekoratyviams — `alt=""`)
- [ ] Form'os — `<label>` kiekvienam input'ui
- [ ] Skip-to-content nuoroda
- [ ] `prefers-reduced-motion` palaikymas animacijoms
- [ ] Ekrano skaitytuvas (NVDA / VoiceOver) — praeina pagrindinį flow

Testavimas: [axe DevTools](https://www.deque.com/axe/devtools/) (Chrome plėtinys) + Lighthouse Accessibility score 95+.

## 11. Saugumas

### OWASP Top 10 kompliansas

| Grėsmė | Apsauga |
|---|---|
| Injection | Prisma ORM (parametrizuotos užklausos automatiškai) |
| Broken Auth | JWT su short-lived access tokens + refresh tokens, httpOnly cookies |
